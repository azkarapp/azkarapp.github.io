# Azkar App

تطبيق أذكار ومسبحة ومهام يومية يعمل كتطبيق ويب تقدمي (PWA).

## المميزات
- أذكار يومية
- مسبحة إلكترونية
- مهام يومية
- إحصائيات يومية
- دعم العمل بدون إنترنت
- حفظ الحالة محليًا

## طريقة التشغيل
1. افتح المشروع عبر خادم محلي
2. افتح `index.html`
3. تأكد من تفعيل Service Worker في بيئة تدعم ذلك

## بنية المشروع
- `index.html`: الهيكل العام للواجهة
- `js/app.js`: منسق التطبيق العام
- `js/storage.js`: إدارة الحالة والتخزين
- `js/content.js`: عرض الصفحة الرئيسية والمحتوى
- `js/masbaha.js`: منطق المسبحة
- `js/tasks.js`: منطق المهام
- `js/tasks/tasks-ui.js`: طبقة عرض المهام
- `js/tasks/tasks-stats.js`: حسابات إحصائيات المهام
- `js/dom.js`: أدوات DOM المشتركة
- `js/pwa.js`: تسجيل وإدارة PWA
- `sw.js`: Service Worker
- `manifest.json`: إعدادات PWA

## الإعدادات
يتم حفظ الإعدادات داخل `storage.state.settings`.

## ملاحظات
- التطبيق يعتمد على localStorage
- يدعم العمل بدون إنترنت بعد تحميله مرة واحدة
